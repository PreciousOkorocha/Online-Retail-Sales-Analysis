# Online-Retail-Sales-Analysis
This is an end-to-end retail sales analysis project using SQL, Jupyter Notebook, and Power BI for data cleaning, EDA, visualization, and business insights.

## Project Overview
This project analyzes an online retail dataset to uncover sales trends, customer purchasing behavior, product performance, and revenue distribution. The project demonstrates the complete data analytics workflow, including data cleaning, exploratory data analysis (EDA), visualization, and insight generation.

## Objectives
- Clean and prepare raw retail transaction data for analysis.
- Identify and handle missing values, duplicates, and invalid records.
- Analyze customer purchasing behavior and revenue patterns.
- Identify top-selling and top revenue-generating products.
- Examine monthly sales trends and country-level revenue performance.
- Generate business insights to support decision-making.

## Tools Used
- SQL Server
- Python (Pandas)
- Jupyter Notebook
- Power BI
- Excel

## Data Cleaning
The following data cleaning tasks were performed:
- Removed records with missing CustomerID and Description values.
- Removed duplicate records.
- Removed invalid Quantity and UnitPrice values.
- Standardized text fields for consistency.
- Validated and converted date fields for time-series analysis.
- Created a cleaned dataset for analysis while preserving the original data.

## Exploratory Data Analysis
The analysis focused on:
- Top Selling Products
- Top Revenue-Generating Products
- Revenue by Country
- Monthly Sales Trends
- Most Valuable Customers
- Customer Purchasing Behavior

## Key Findings
- The United Kingdom generated the highest revenue among all countries.
- Revenue increased significantly during the final quarter of the year, indicating seasonal demand patterns.
- A small number of products contributed a large share of total revenue.
- Customer spending was concentrated among a relatively small group of high-value customers.
- The dataset exhibited a Pareto distribution, where a minority of customers and products generated most revenue.
